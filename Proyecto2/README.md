## Negociación Técnica en Equipos de Desarrollo de Software
 
## Descripción del proyecto
Este repositorio contiene el desarrollo del Proyecto 2 del curso Comunicación Asertiva, enfocado en la aplicación de técnicas de comunicación, análisis y negociación dentro de un contexto real de negocios.

El proyecto se desarrolló en dos fases principales:

## Fase 1: Recolección de información
Se realizaron entrevistas a dos negocios independientes ubicados en el mercado La Reformita. El equipo se dividió para cubrir ambas entrevistas (3 integrantes por negocio).

A partir de esta fase se generó:
- Recolección de evidencias
- Justificación del problema
- Listado de preguntas utilizadas en la entrevista
- Elevator pitch individual basado en la entrevista asignada

## Fase 2: Análisis y propuesta
Se seleccionó una de las entrevistas realizadas (entrevista 2) como base para el desarrollo de la solución.

En esta fase se trabajó en:
- Análisis del flujo del negocio
- Identificación de requisitos
- Propuesta de solución
- Simulación de una junta directiva para la negociación de la solución
- Evaluación final (Post-Mortem)

## Objetivo
Aplicar principios de comunicación asertiva para analizar problemas reales, estructurar propuestas y negociar soluciones de manera clara, efectiva y profesional.
